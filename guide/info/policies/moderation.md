# Moderation Policies
A list of our current moderation policies.

### Punish All Members Involved
If a fight breaks out or drama ensues between multiple members, punish all members involved.
There will be no "they started it" bullshit, if you participated, you get punished.
If there is a clear and obvious instigator, they should get punished worse then the rest of the members involved.

### Double Warn for Severe Infractions
If a member commits a severe infraction that's unworthy of another punishment,
the member should be warned twice for it.
This puts the member at strike 2, meaning one more infraction is a ban.
When issuing the second warning, add "(second warn, severity)" to the end of the warn message.

### Three Warns = Ban
If a member stacks up 3 warns within 6 months, they should be banned with the chance to appeal.
Sapphire will automatically send an alert in staff lounge, and a moderator or admin can take the case and ban the member.
