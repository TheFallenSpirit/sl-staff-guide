# ES Flagging Guide
A guide to flagging users on Eternal Slave.

### Criteria
There are 3 reasons that a user can be flagged on ES.

- **Minor** - If the user is a minor, either they select a minor role,
admit to being a minor, or have a minor age in their bio.

- **Findom** - If the user is a financial dominant,
either they must admit to being a findom, or have findom related info in their bio.

- **Seller** - If the user account is a bot account specifically for selling content.
These accounts are typically called "mega seller" or "mega link", but many other instances also exist.

### How to Flag
To flag a user on ES, use `/safetyctl flag @user <flag> <reason>` and attach 1-5 screenshots as evidence of the infraction.
The "flag" and "reason" options will autocomplete, and are multiple choice.

After sending the command, it will load for up to 15 seconds before showing a confirmation prompt.
The prompt will display the flag, target user, and reason.
You will need to wait 5 seconds before confirming, then tap/click confirm to proceed with the flag.

After the flag succeeds, the target user will be banned from the server and our partner network if applicable.

:::info
If you have any questions regarding flagging, please direct them to an admin or higher.
:::
